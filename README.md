 # Encabezados   
 Hacer siempre # para generar titulos  
 ## titulo 2  
 ### titulo 3

 # formato de texto  
 negrita: **texto** : son dos **  
 cursiva: *texto* : solo un *
 negrita y cursiva: ***texto***  : son 3 ***
 Tachado: ~~texto~~(para hacer este simbolo es: option + *)  
 escapado de simbolos: \#texto : option + signo de pregunta para \
